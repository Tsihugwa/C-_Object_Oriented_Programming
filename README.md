# C-_Object_Oriented_Programming
LoginPage
This is a simple login page created in C# using Windows Forms. It has a username and password textbox for user input and a submit button. The code checks to see if the input matches the predetermined username and password and if it does, it opens the BookingPage form. If the input is incorrect, it displays a message telling the user to input the correct username or password.

Getting Started
To use this code, you will need to have Visual Studio installed and create a Windows Forms project. Add a form to your project and copy the code above into the form's code file. Run the program and test the login feature by inputting the predetermined username and password or incorrect input to see the appropriate response.

Prerequisites
Visual Studio
C# knowledge

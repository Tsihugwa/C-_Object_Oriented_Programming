# C-_Object_Oriented_Programming
LoginPage
This is a simple login page created in C# using Windows Forms. It has a username and password textbox for user input and a submit button. The code checks to see if the input matches the predetermined username and password and if it does, it opens the BookingPage form. If the input is incorrect, it displays a message telling the user to input the correct username or password.

Getting Started
To use this code, you will need to have Visual Studio installed and create a Windows Forms project. Add a form to your project and copy the code above into the form's code file. Run the program and test the login feature by inputting the predetermined username and password or incorrect input to see the appropriate response.

BookingPage
This is a booking page for a car rental company. It allows users to select their preferred car type, fuel type, duration of rental, and optional preferences such as unlimited mileage and breakdown cover. The total cost of the booking is then calculated and displayed to the user.

Features
Exit button to close the program
Summary button to display the total cost of the booking
Save button to save the booking information
How to Use
Fill in the required information such as first name, surname, address, and age.
Select whether or not you have a driving license.
Choose your preferred car type from the dropdown list.
Select your preferred fuel type from the dropdown list.
Choose the duration of your rental from the dropdown list.
Select any optional preferences you would like to include in your booking.
Click the summary button to view the total cost of your booking.
If you want to save your booking information, click the save button.
To exit the program, click the exit button.
Dependencies
This program uses the System.Windows.Forms namespace and the MessageBox class.

Future Improvements
Add a confirmation page before exiting the program
Allow users to input their own car and fuel preferences if they are not listed in the dropdown menus
Implement a payment system to allow users to make their bookings directly through the program.
